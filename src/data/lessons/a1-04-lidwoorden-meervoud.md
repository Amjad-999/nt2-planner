# أدوات التعريف والجمع — Lidwoorden & meervoud

## de / het / een

- **de** و**het** = «الـ» (معرفة). معظم الكلمات *de*، وبعضها *het* (تُحفَظ).
- **een** = نكرة (a/an).
- كلّ الأسماء في **الجمع** تأخذ *de*.

| الأداة | الاستخدام | مثال |
|---|---|---|
| de | أغلب الكلمات + كلّ الجمع | *de man، de boeken* |
| het | مجموعة تُحفَظ | *het huis، het kind* |
| een | نكرة مفردة | *een man، een huis* |

---

## الجمع بـ -en (الأكثر شيوعًا)

| مفرد | جمع |
|---|---|
| boek | boeken |
| man | mannen |
| boom | bomen |
| huis | huizen |
| brief | brieven |

> 💡 تغيّرات إملائية: مضاعفة الساكن (*man → mannen*)، تبسيط العلّة (*boom → bomen*)، و`s→z` / `f→v` (*huis → huizen*، *brief → brieven*).

---

## الجمع بـ -s و 's

نضيف `-s` بعد `-el، -em، -en، -er، -e`، ونضيف `'s` بعد حروف العلّة a / o / u / i:

| مفرد | جمع |
|---|---|
| tafel | tafels |
| meisje | meisjes |
| foto | foto's |
| auto | auto's |

---

## جموع شاذّة

| مفرد | جمع | المعنى |
|---|---|---|
| kind | kinderen | طفل / أطفال |
| ei | eieren | بيضة / بيض |
| stad | steden | مدينة / مدن |

> ⚠️ كلّ اسم في الجمع يأخذ **de**: *het kind → de kinderen*.
