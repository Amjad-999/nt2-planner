# الفعلان hebben و zijn

الفعلان *hebben* (يملك / لديه) و*zijn* (يكون) من أكثر الأفعال استخدامًا، وهما **شاذّان** — احفظ تصريفهما جيّدًا.

---

## hebben — يملك / لديه

| الفاعل | الصيغة |
|---|---|
| ik | **heb** |
| jij / je | **hebt** |
| u | hebt / heeft |
| hij / zij / het | **heeft** |
| wij / jullie / zij | **hebben** |

- *Ik heb een fiets.* — لديّ درّاجة.
- *Zij heeft twee kinderen.* — لديها طفلان.

---

## zijn — يكون

| الفاعل | الصيغة |
|---|---|
| ik | **ben** |
| jij / je | **bent** |
| u | bent |
| hij / zij / het | **is** |
| wij / jullie / zij | **zijn** |

- *Ik ben moe.* — أنا متعب.
- *Hij is ziek.* — هو مريض.
- *Wij zijn thuis.* — نحن في البيت.

> 💡 نستخدم **zijn** للحال والمكان والصفة، و**hebben** للملكية.
