# اسم الفاعل واسم المفعول كصفة — Deelwoorden als adjectief

## اسم الفاعل (tegenwoordig deelwoord)

**المصدر + -d**، ويُستعمل صفةً (ويأخذ -e كالصفات):

- *slapen → **slapend***: *de **slapende** baby* — الطفل النائم
- *lachen → **lachend***: *een **lachend** kind* — طفل ضاحك

---

## اسم المفعول (voltooid deelwoord) صفةً

- *sluiten → gesloten*: *de **gesloten** deur* — الباب المغلق
- *koken → gekookt*: *een **gekookt** ei* — بيضة مسلوقة
- *breken → gebroken*: *het **gebroken** glas* — الزجاج المكسور

> 💡 كلاهما يأتي **قبل الاسم** ويتبع قاعدة -e: *een gekookt ei* (بلا -e لأنّها كلمة het نكرة) مقابل *de gekookte eieren* (بـ -e).
