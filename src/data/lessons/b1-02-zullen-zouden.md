# zullen و zouden

## zullen — المستقبل والاقتراح والوعد

| الفاعل | الصيغة |
|---|---|
| ik | zal |
| jij | zult / zal |
| hij / zij / het | zal |
| wij / jullie / zij | zullen |

- *Ik **zal** je helpen.* — سأساعدك (وعد).
- ***Zullen** we gaan?* — هل نذهب؟ (اقتراح)
- *Het **zal** morgen regenen.* — ستمطر غدًا.

---

## zouden — الشرطي والتأدّب

| الفاعل | الصيغة |
|---|---|
| ik / hij / zij | zou |
| wij / jullie / zij | zouden |

- *Ik **zou** graag een koffie willen.* — أودّ قهوة (تأدّب).
- *Als ik rijk **was**, **zou** ik reizen.* — لو كنت غنيًّا لسافرت.
- ***Zou** je me kunnen helpen?* — هل يمكنك مساعدتي؟

> 💡 للمستقبل يكثر أيضًا استخدام **gaan**: *Ik **ga** morgen werken.*
