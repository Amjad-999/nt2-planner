# أسماء الإشارة — Aanwijzende voornaamwoorden

تشير إلى شيء قريب أو بعيد، وتتبع نوع الكلمة (de / het).

| | قريب (هذا) | بعيد (ذاك) |
|---|---|---|
| كلمة **de** | **deze** | **die** |
| كلمة **het** | **dit** | **dat** |
| الجمع (دائمًا de) | **deze** | **die** |

- ***deze** man* (هذا الرجل) · ***die** vrouw* (تلك المرأة)
- ***dit** huis* (هذا البيت) · ***dat** boek* (ذلك الكتاب)
- ***deze** kinderen* (هؤلاء الأطفال)

> 💡 المفرد het → **dit / dat**، وكلّ ما عداه (de + الجمع) → **deze / die**.

- *Is **dit** jouw tas?* — هل هذه حقيبتك؟
- *Ik wil **die** schoenen kopen.* — أريد شراء تلك الأحذية.
