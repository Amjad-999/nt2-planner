# الضمائر الشخصية — Persoonlijke voornaamwoorden

تحلّ محلّ الأسماء. لها صيغة **فاعل** (onderwerp) وصيغة **مفعول** (lijdend voorwerp).

---

## ضمائر الفاعل (onderwerp)

| الضمير | المعنى |
|---|---|
| ik | أنا |
| jij / je | أنتَ (غير رسمي) |
| u | أنتَ (رسمي) |
| hij | هو |
| zij / ze | هي |
| het | هو/هي (للأشياء het) |
| wij / we | نحن |
| jullie | أنتم |
| zij / ze | هم |

---

## ضمائر المفعول (lijdend voorwerp)

| الضمير | المعنى |
|---|---|
| mij / me | ـني |
| jou / je | ـك |
| u | ـك (رسمي) |
| hem | ـه |
| haar | ـها |
| het | ـه (للأشياء) |
| ons | ـنا |
| jullie | ـكم |
| hen / ze | ـهم |

- *Ik zie hem.* — أراه.
- *Zij helpt mij.* — هي تساعدني.
- *Wij kennen hen.* — نعرفهم.

> 💡 **je** صيغة غير منبورة من *jij* (فاعل) ومن *jou* (مفعول)، و**u** للاحترام والمواقف الرسمية.
