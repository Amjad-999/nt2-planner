# er و daar

**er** كلمة متعدّدة الوظائف، و**daar / hier** نسختها المنبورة.

| الوظيفة | مثال |
|---|---|
| المكان | *Ik woon **er** al jaren.* |
| العدد / الكمّية | *Ik heb **er** drie.* |
| فاعل مؤقّت | ***Er** staat een man bij de deur.* |
| مع حرف جرّ | *Ik denk **er**aan / **er**over.* |
| المبني للمجهول | ***Er** wordt gedanst.* |

---

## er أم daar؟

- **er + حرف الجرّ** (غير منبور): *Ik reken **erop**.*
- **daar + حرف الجرّ** (منبور / للتأكيد أو الإشارة): ***Daar** heb ik niet aan gedacht.*

> 💡 يمكن فصل er/daar عن حرف الجرّ: *Ik denk **er** niet **aan**.* — *Daar denk ik niet **aan**.*
