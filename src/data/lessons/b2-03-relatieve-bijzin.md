# جمل الوصل (متقدّم) — Relatieve bijzinnen

جملة الوصل **bijzin** فالفعل في النهاية، وتبدأ بضمير وصلٍ يناسب الاسم.

| يعود على | الضمير |
|---|---|
| كلمة de / جمع | **die** |
| كلمة het | **dat** |
| alles / iets / niets / الجملة كلّها | **wat** |

- *De collega **die** naast me **zit**…*
- *Het rapport **dat** ik **schreef**…*

---

## مع حرف الجرّ

- **للأشياء:** `waar + حرف الجرّ`:
  - *de stoel **waarop** ik zit* (أو: *waar ik **op** zit*)
  - *het onderwerp **waarover** we praten*
- **للأشخاص:** `حرف الجرّ + wie`:
  - *de man **met wie** ik werk*
  - *de vrouw **voor wie** ik dit doe*

> 💡 **wat** تشير أيضًا إلى الجملة كاملةً: *Hij was te laat, **wat** ik vervelend vond.*
