# ضمائر الوصل — Betrekkelijke voornaamwoorden

تربط جملةً وصفية بالاسم (= الذي / التي). الجملة الموصولة **bijzin** فيذهب الفعل إلى النهاية.

| يعود على | الضمير | مثال |
|---|---|---|
| كلمة **de** / جمع | **die** | *De man **die** daar **staat**.* |
| كلمة **het** | **dat** | *Het boek **dat** ik **lees**.* |
| alles / iets / niets / dat | **wat** | *Alles **wat** ik **weet**.* |

- *Dit is de vrouw **die** in Amsterdam **woont**.*
- *Waar is het pakje **dat** gisteren **kwam**?*

---

## مع حرف جرّ

- للأشياء: **waar + حرف الجرّ**: *de stoel **waarop** ik zit*.
- للأشخاص: **حرف الجرّ + wie**: *de man **met wie** ik praat*.

> 💡 لاحظ ذهاب الفعل إلى نهاية جملة الوصل (staat، lees، woont).
