# جمل الشرط والزمن — "als"

**als** أداة تبعية تعني «إذا / عندما»، وتُرسل الفعل إلى **نهاية** جملتها.

- *Als het **regent**, blijf ik thuis.* — إذا أمطرت، أبقى في البيت.
- *Ik bel je **als** ik klaar **ben**.* — سأتّصل بك عندما أنتهي.

> 💡 إذا بدأت الجملة بـ **als…**، تبدأ الجملة الرئيسية بعدها بالفعل (قلب): *Als het regent, **blijf ik** thuis.*

---

## als / wanneer / toen

| الأداة | الاستخدام |
|---|---|
| **als** | شرط أو حدث متكرّر (حاضر/مستقبل) |
| **wanneer** | «متى» (سؤال) أو بمعنى als |
| **toen** | حدث **واحد** في الماضي |

- ***Toen** ik klein was, woonde ik in een dorp.* — عندما كنت صغيرًا، سكنت في قرية.
