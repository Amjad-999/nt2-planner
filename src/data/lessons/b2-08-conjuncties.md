# أدوات الربط المتقدّمة — Voegwoorden (B2)

أدوات **تبعية** تُرسل الفعل إلى النهاية (عدا أدوات التنسيق: en/maar/want/of/dus).

| المعنى | الأداة | مثال |
|---|---|---|
| النتيجة | **zodat** | *Hij sprak luid **zodat** iedereen het **hoorde**.* |
| الغرض | **opdat** | *…**opdat** er geen fouten **ontstaan**.* |
| الشرط | **mits** / **tenzij** | *…**tenzij** het **regent**.* |
| التنازل | **hoewel / ofschoon** | ***Hoewel** hij ziek **was**, kwam hij.* |
| السبب | **omdat / doordat / aangezien** | *De weg is glad **doordat** het **vriest**.* |
| الزمن | **zodra / totdat / nadat / voordat** | ***Zodra** ik klaar **ben**, bel ik.* |
| التناسب | **naarmate** | ***Naarmate** je ouder **wordt**, leer je meer.* |

> 💡 فرق دقيق: **omdat** (سبب/دافع) مقابل **doordat** (سبب مباشر بلا إرادة): *Hij viel **doordat** de vloer nat was.*
