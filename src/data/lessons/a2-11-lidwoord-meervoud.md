# التعريف والجمع (تعمّق A2)

مراجعة وتوسعة لمستوى A1.

---

## التصغير (verkleinwoord) → دائمًا het + جمع -s

- *het huis → het **huisje*** → de **huisjes**
- *de man → het **mannetje*** → de **mannetjes**

> 💡 كلّ اسم مصغّر (‎-je / -tje / -pje / -etje) هو **het**، وجمعه دائمًا **-s**.

---

## جموع خاصّة

| مفرد | جمع | ملاحظة |
|---|---|---|
| de stad | de **steden** | تغيّر داخلي |
| het kind | de **kinderen** | + eren |
| de mogelijkheid | de **mogelijkheden** | ‎-heid → -heden |
| het museum | de **musea** | كلمة لاتينية |
| de baby | de **baby's** | ‎'s بعد حرف علّة |

---

## de أم het؟ (تذكير)

- المصغّرات وأغلب كلمات ‎-um / -ment → **het**.
- الأشخاص وأغلب كلمات ‎-ing / -heid / -tie، وكلّ الجمع → **de**.

> ⚠️ لا قاعدة قاطعة — احفظ الأداة مع كل كلمة جديدة.
