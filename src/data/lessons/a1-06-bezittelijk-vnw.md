# ضمائر الملكية — Bezittelijke voornaamwoorden

تأتي قبل الاسم وتدلّ على المالك.

| الضمير | المعنى | مثال |
|---|---|---|
| mijn | ـي | *mijn huis* |
| jouw / je | ـك | *jouw boek* |
| uw | ـك (رسمي) | *uw naam* |
| zijn | ـه | *zijn auto* |
| haar | ـها | *haar tas* |
| ons / onze | ـنا | *ons huis / onze auto* |
| jullie | ـكم | *jullie school* |
| hun | ـهم | *hun kinderen* |

- *Dat is mijn fiets.* — تلك درّاجتي.
- *Hoe is jouw naam?* — ما اسمك؟
- *Haar moeder werkt hier.* — أمّها تعمل هنا.

---

## ons أم onze؟

| | het-woord (مفرد) | de-woord / جمع |
|---|---|---|
| نحن (ملكنا) | **ons** — *ons huis* | **onze** — *onze auto، onze kinderen* |

> ⚠️ **zijn** قد تكون ملكية («ـه») وقد تكون الفعل «يكون» — حسب السياق.
