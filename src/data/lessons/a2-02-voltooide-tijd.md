# الماضي التام — Perfectum (v.t.t.)

الأكثر استخدامًا للكلام عن أحداث منتهية.

**الصيغة: فعل مساعد (hebben / zijn) + اسم المفعول في نهاية الجملة.**

```
Ik   heb   gisteren   gewerkt.
     مساعد              اسم المفعول
```

---

## اسم المفعول (voltooid deelwoord)

- **منتظم:** `ge-` + الجذع + **-t / -d** (حسب 't kofschip): werken → **gewerkt**، wonen → **gewoond**.
- **شاذّ:** `ge-…-en` ويُحفظ: schrijven → **geschreven**، zien → **gezien**، gaan → **gegaan**.

> 💡 الأفعال ببادئة be-, ver-, ge-, ont-, her- **لا** تأخذ ge-: *betalen → betaald*، *vertellen → verteld*.

---

## hebben أم zijn؟

| المساعد | متى؟ | مثال |
|---|---|---|
| **zijn** | حركة أو تغيّر مكان/حال | *Ik ben naar huis gegaan.* |
| **hebben** | باقي الأفعال (الأغلب) | *Ik heb gegeten.* |

- *Wij hebben hard gewerkt.* — عملنا بجدّ.
- *Zij is gisteren gekomen.* — أتت أمس.
- *Heb je de brief geschreven?* — هل كتبتَ الرسالة؟
