# أفعال مع حروف جرّ ثابتة — Vaste voorzetsels

كثير من الأفعال يرتبط بحرف جرّ **ثابت** يجب حفظه معه.

| الفعل + الحرف | المعنى | مثال |
|---|---|---|
| denken **aan** | يفكّر في | *Ik denk **aan** jou.* |
| wachten **op** | ينتظر | *We wachten **op** de bus.* |
| houden **van** | يحبّ | *Ik houd **van** muziek.* |
| kijken **naar** | يشاهد | *Hij kijkt **naar** de film.* |
| luisteren **naar** | يستمع إلى | *Luister **naar** mij.* |
| vragen **om** | يطلب | *Ze vraagt **om** hulp.* |
| beginnen **met** | يبدأ بـ | *We beginnen **met** les 1.* |
| bang zijn **voor** | يخاف من | *Hij is bang **voor** honden.* |

> 💡 عند الإحالة لشيء نَدمج **er + الحرف**: *Ik denk **eraan**.* — أفكّر فيه.
