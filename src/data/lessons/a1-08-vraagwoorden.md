# أدوات الاستفهام — Vraagwoorden

نبدأ بها الأسئلة المفتوحة، ويأتي بعدها **الفعل ثمّ الفاعل**.

| الأداة | المعنى | مثال |
|---|---|---|
| wie | مَن | *Wie ben jij?* |
| wat | ماذا | *Wat doe je?* |
| waar | أين | *Waar woon je?* |
| wanneer | متى | *Wanneer kom je?* |
| hoe | كيف | *Hoe gaat het?* |
| waarom | لماذا | *Waarom leer je Nederlands?* |
| welke / welk | أيّ | *Welke kleur?* |
| hoeveel | كم (عدد) | *Hoeveel kinderen?* |
| hoe laat | كم الساعة | *Hoe laat is het?* |

- *Waar woon jij?* — أين تسكن؟
- *Hoe heet je?* — ما اسمك؟
- *Wat eet je?* — ماذا تأكل؟

> 💡 **welk** مع كلمات *het*، و**welke** مع كلمات *de* والجمع: *welk huis? / welke auto?*
