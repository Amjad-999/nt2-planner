# مفعول حرف الجرّ المتقدّم — Vaste voorzetsels (B2)

أفعالٌ وصفاتٌ ترتبط بحرف جرّ ثابت يجب حفظه.

| التعبير | المعنى |
|---|---|
| rekenen **op** | يعتمد على |
| zich ergeren **aan** | ينزعج من |
| deelnemen **aan** | يشارك في |
| twijfelen **aan** | يشكّ في |
| zich interesseren **voor** | يهتمّ بـ |
| verlangen **naar** | يتوق إلى |
| zorgen **voor** | يعتني بـ |
| beschikken **over** | يتوفّر لديه |
| lijden **aan** | يعاني من |

---

## مع er والجملة الثانوية

- *Ik **reken erop**.* — أعتمد عليه.
- *Ik reken **erop dat** je op tijd **komt**.* — أعتمد على أنّك ستأتي في الوقت.

> 💡 عند الإحالة إلى شيء: **er / daar + حرف الجرّ** (erop، eraan، ervoor…).
