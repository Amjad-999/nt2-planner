# الصفة ونهاية -e — Het adjectief

الصفة التي تأتي **قبل الاسم** تأخذ **-e** عادةً، إلّا في حالة واحدة.

---

## القاعدة

| الحالة | ‎-e؟ | مثال |
|---|---|---|
| كلمة *de* (مفرد) | نعم | de **grote** man / een **grote** man |
| كلمة *het* + the/this | نعم | het **grote** huis / dit **grote** huis |
| كلمة *het* + **een** (نكرة مفرد) | **لا** | een **groot** huis |
| الجمع (دائمًا) | نعم | **grote** huizen |

> ⚠️ الاستثناء الوحيد: **een + كلمة het مفردة** → بلا -e: *een groot huis، een klein kind، een mooi boek*.

---

## الصفة بعد الفعل (خبرية)

إذا جاءت بعد *zijn / worden* (وليست قبل اسم) → **بلا -e**:

- *Het huis is **groot**.* — البيت كبير.
- *De bloemen zijn **mooi**.* — الزهور جميلة.
